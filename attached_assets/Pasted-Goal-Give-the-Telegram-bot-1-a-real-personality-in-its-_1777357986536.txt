Goal
Give the Telegram bot (1) a real personality in its answers — not just in alerts — and (2) the autonomy to form and update its own opinions about any subject that comes up in conversation, with no human curation.

Voice — Don / Wife / Son in every Telegram answer

The same three-character ensemble previously used only for Constellation alerts (calm strategist Don, sharp strategist Wife, activist Son) is now the bot's everyday answering voice.
The Don leads every reply; the other two chime in only when their angle adds something distinct.
The persona is delivered through the model's high-priority instructions channel, so a long PDF or a leading question can't easily talk it out of character.
The prompt explicitly tells it to take a stance rather than report neutrally — that's the substrate the opinion loop below feeds on.
Autonomous opinions — the new "Opinions" surface

New database table for autonomously-formed beliefs about any subject (separate from the existing source-anchored beliefs).
Each opinion has: subject, current stance, confidence (0..1), how many times evidence has accumulated, the latest supporting evidence, the channel it last came from (typed Q&A or PDF), and a rolling history of the last 10 movements.
A small extractor pass runs after every Telegram answer (both typed questions and PDF reads). It pulls up to five {subject, stance, confidence, evidence} records per exchange and upserts them.
Runs detached from the reply, so it never delays your message and a failure can never break the answer.
Cheap-and-quiet guards skip slash-commands, very short answers, and apology refusals.
Concurrent updates to the same subject are serialized by a row lock; a one-time retry handles the rare race where two first-time inserts collide.
New API + UI

New endpoint to fetch the opinions list, with proper validation (bad query parameters return a clean 400).
New "Opinions" tab in the Ephemeroi sidebar (Brain icon, between Beliefs and Tensions) showing each subject in The Don's serif quote style with: stance, confidence percentage, a trend arrow when the latest movement differs from the previous one, evidence count, and the channel that last touched it.
The page refreshes every 15 seconds, so you can watch opinions form in near real time as you chat with the bot.
For privacy, the raw question text is kept in the database for future use but intentionally not returned over the public API surface.
